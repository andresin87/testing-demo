# Steps

### Create a react app
```
npx create-react-app .
```

### Install prettier
```
npm i -D prettier
```

### Add prettier config file

`.prettierrc` file

```
{
  "singleQuote": true
}
````

...and add the prettier script on package.json npm scripts

### Add storybook

```
npx sb init
```

### Run project and storybook
```
npm run start & npm run storybook
```
